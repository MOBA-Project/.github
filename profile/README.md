# Mova Movies

경량 영화 검색 · 북마크 · 리뷰 웹앱

> 관심 있는 영화를 빠르게 찾고, 북마크로 관리하며, 간단한 리뷰로 기록하는 개인 큐레이션 서비스<br />

* **`Team name` :** Mova

## 핵심 기능

* **`로그인 및 메인페이지` :** 히어로 슬라이드를 통해 시각적으로 강조된 배너 구성 및 주요 영화, 이슈 콘텐츠 섹션화
<br/>
![demo](https://github.com/user-attachments/assets/ff809fb7-8873-46f6-9c1a-88cbe9645680)
<br/>

* **`영화 리스트 및 조회` :** 영화 검색 및 장르별로 렌더링, 북마크 시각화
<br/>
![demo2](https://github.com/user-attachments/assets/5d783fe5-f70d-4bf5-bb61-52e0fb47a96c)
<br/>
* **`영화 상세 정보 및 댓글(커뮤) 기능` :** 영화 상세 페이지에서 배우 정보와 연관 영화 리스트를 한눈에 확인하고,
댓글로(좋아요) 다른 유저와 의견을 나눌 수 있어요.
<br/>
![demo3](https://github.com/user-attachments/assets/e1da9b15-ffe1-49c3-b182-a4bfe4d62b24)
<br/>
* **`AI 추천 시스템` :** 사용자의 영화 취향과 데이터를 기반으로 OpenAI API를 활용한 영화 추천 기능을 제공합니다.
<br/>
![demo4](https://github.com/user-attachments/assets/fcaba4f8-c925-4c96-8ec6-3f0d7fc3d5a6)
<br/>

* **`마이 리스트 페이지` :** 마이 리스트 페이지에서 자신이 북마크한 영화와 리뷰를 남긴 영화, AI 추천을 받은 영화 리스트를 한 번에 보실 수 있습니다.

![demo5](https://github.com/user-attachments/assets/03afbc43-18a1-48c5-90d3-65c92e415096)


* **`영화 감상문 작성` :** 영화에 대한 감상문을 작성하고 댓글을 달 수 있습니다.
